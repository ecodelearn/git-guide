Instalação do Git

Siga as instruções abaixo para instalar o Git no seu sistema operacional:

Windows

  1. Acesse o site oficial do Git: Git para Windows
  2. Baixe o instalador e execute-o.
  3. Siga as instruções do instalador, mantendo as opções padrão.
  4. Após a instalação, abra o Prompt de Comando e digite git --version para verificar se a instalação foi bem-sucedida.

macOS

  1. Abra o Terminal.
  2. Instale o Homebrew (se ainda não tiver) com o seguinte comando:
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  3. Após a instalação do Homebrew, instale o Git com o comando:
    brew install git
  4. Verifique a instalação digitando git --version no Terminal.

Linux (Ubuntu/Debian)

  1. Abra o Terminal.
  2. Atualize os pacotes do sistema:
    sudo apt update
  3. Instale o Git:
    sudo apt install git
  4. Verifique a instalação digitando git --version no Terminal.
